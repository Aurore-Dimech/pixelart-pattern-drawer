# v1.plan.md — Plan contractuel PixelArt App

**Statut** : ✅ Implémenté  
**Date de création** : phase initiale  
**Périmètre** : MVP fonctionnel couvrant auth + éditeur + sauvegarde + galerie + favoris

---

## Objectif de la v1

Livrer une application web fonctionnelle où un utilisateur peut :
1. Créer un compte et se connecter
2. Créer un dessin pixel art dans un éditeur interactif
3. Sauvegarder ses dessins
4. Publier un dessin dans la galerie publique
5. Consulter la galerie et ajouter des dessins en favoris

---

| F9 | Export PNG du dessin | 🟡 Nice-to-have |

## Périmètre exclu (OUT OF SCOPE v1)

- Suggestion de palette par IA (v2)
- Commentaires sur les dessins
- Tags / catégories (v2)
- Profil public utilisateur
- Système de likes / votes
- Animations pixel art

## Schéma Prisma implémenté

Le schéma final inclut les tables Tags (ajoutées en v2) :

```prisma
model User { id, email, name, passwordHash, createdAt, drawings[], favorites[] }
model Drawing { id, title, gridData, isPublished, createdAt, updatedAt, authorId, tags[], favorites[] }
model Tag { id, name, slug, drawings[] }
model DrawingTag { drawingId, tagId }  -- PK composite
model Favorite { userId, drawingId }   -- unique(userId, drawingId)
```
